```typescript
"tool.execute.before": async (input, output) => {
  const tool = String(input?.tool ?? "").toLowerCase()
  if (tool !== "bash" && tool !== "shell") return

  // Only rewrite for GPT/Gemini family models
  if (!isGptOrGeminiFamily(currentModel)) {
    return  // Pass through for non-GPT/Gemini models
  }

  const args = output?.args
  if (!args || typeof args !== "object") return
  // ... rest of existing logic
}
```

### Behavior Logic Flow

```
tool.execute.before called
  ↓
Check tool type (bash/shell)
  ↓
Check isGptOrGeminiFamily(currentModel)
  ↓
├─ false → return early (pass through unchanged)
└─ true  → proceed with rtk rewrite logic
```

### Expected Behavior

1. **Unknown model** (`currentModel = null`):
   - `isGptOrGeminiFamily(null)` returns `false`
   - Commands pass through unchanged
   - Conservative default: no rewriting

2. **GPT family models** (OpenAI/Azure/gpt-*/o1-*/o3-*/codex-*/codex):
   - `isGptOrGeminiFamily()` returns `true`
   - Commands rewritten via `rtk rewrite`
   - Token savings enabled

3. **Gemini family models** (Google/gemini/vertex/gemini-*):
   - `isGptOrGeminiFamily()` returns `true`
   - Commands rewritten via `rtk rewrite`
   - Token savings enabled

4. **Other models** (Claude, Qwen, GLM, etc.):
   - `isGptOrGeminiFamily()` returns `false`
   - Commands pass through unchanged
   - No rewriting attempted
